# Gigih Play
#### Author: [Teza Alfian](https://github.com/tezaalfian)

## Description
This API is used to manage the data of Gigih Play (Tokopedia Play Clone) application. This API is built using NodeJS and ExpressJS framework. This API is also connected to MongoDB.

## API Documentation
### Videos
#### Get All Videos
- Method: GET
- Endpoint: `/videos`
- Response Body Success :
```json
{
    "data": [
        {
            "id": "string",
            "title": "string",
            "thumbnail": "string",
            "description": "string",
            "urlEmbed": "string",
        }
    ]
}
```

#### Get Video By ID
- Method: GET
- Endpoint: `/videos/:id`
- Response Body Success :
```json
{
    "data": {
        "id": "string",
        "title": "string",
        "thumbnail": "string",
        "description": "string",
        "urlEmbed": "string",
    }
}
```
- Response Body Error :
```json

        "string"
    ]
}
```
